CMPSCI 687: Course Project Fall 2020

Submission contains 100 text files that are named as policyX.txt for X ∈ {1, 2, . . . , 100}. 
The directory named “source” contains all the code. 

To run the code enter the following command:

Command
python main.py

P.S: Policies have been generated using the following seeds:
1. A batch of 20 policies with seed = 333 and random seed in CMAE
2. A batch of 25 policies with seed = 333 universally
3. A batch of 20 policies with seed = 222 universally
4. A batch of 20 policies with seed = 111 universally
5. A batch of 10 policies with seed = 444 universally
